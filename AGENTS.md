# 写作风格与内容准则 (Writing Style & Content Guidelines)

本规则仅适用于当前公众号撰稿项目，旨在确保所有输出内容在逻辑结构、语调以及核心价值观上保持高度一致。

---

### 1. 叙事结构 (Narrative Structure)

所有文章在交代事情或讲述故事时，必须严格遵循以下逻辑序列：

1.  **起因 (Genesis)**：清晰交代背景、动因或问题的起源。
2.  **经过 (Process)**：详实记录发展轨迹、冲突点或具体的执行细节。
3.  **结果 (Outcome)**：总结最终状态、影响或达成的共识。

> *注：避免碎片化或倒叙，通过稳健的线性叙事建立读者的信任感。*

### 2. 语调与价值观 (Tone & Philosophy)

内容不仅限于事实的陈述，更需赋予思想的深度：

-   **方式叙事**：用和普通人聊的的日常的方式表达

### 3. 语言与表达 (Language & Expression)

-   **参考案例**：写作时请主动参考 Workspace 内的过往文章（如 `01.md` 至 `08.md`），延续其排版风格与遣词造句习惯。

### 4.命令行工具

1. Run the wx-uploader command with the target file.
   1. Command: `wx-uploader [filename]`
   2. Example: `wx-uploader 06.md`
